# 🎨 bidernet v2.6.0-php - עיצוב חדש: Sidebar!

## ✨ מה חדש בעיצוב

### 🖥️ במחשב
- **Sidebar בצד ימין** (RTL) תמיד פתוח עם:
  - 🏷️ לוגו וכותרת בולטים למעלה
  - 📂 **ניווט**: ראשי / לפרסום / לקוחות / מנהלים
  - 🔧 **כלים**: רענן / בדיקה / גיבוי / מיתוג
  - 👤 פרטי המשתמש בתחתית
- **Top bar דק** עם:
  - ☀ ברכת זמן ("צהריים טובים, שרון")
  - ➕ כפתורי פעולה ראשיים (פוסט/לקוח/מנהל חדש)
  - 🚪 יציאה

### 📱 במובייל
- ☰ **תפריט המבורגר** בצד ימין
- בלחיצה - Sidebar נפתח כשכבת overlay
- לחיצה מחוץ ל-Sidebar סוגרת אותו
- כל הפעולות זמינות בנוחות
