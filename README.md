# 🛠️ bidernet v2.9.7-php - תיקון 2 בעיות

## 🔧 הבעיות שתוקנו

### 1️⃣ פונט הפוסטים לא משתנה
**הבעיה:** כשמשתמש כותב תוכן בעורך הטקסט הויזואלי, הוא לפעמים מצרף inline styles עם פונט אחר (`<span style="font-family: Arial">`) שדורסים את הפונט שהגדרת במערכת.

**התיקון:** הוספתי CSS חזק יותר שמכריח את הפונט שלך **על כל אלמנט במערכת**, אפילו אם יש לו inline style של פונט אחר.

### 2️⃣ QuotaExceededError - localStorage מלא
**הבעיה:** היה גיבוי אוטומטי של כל הפוסטים (כולל תמונות Base64 שלמות) ל-localStorage של הדפדפן. ה-localStorage מוגבל ל-5-10MB, אבל כל תמונה לוקחת 1-5MB. אז בכל שמירה הדפדפן זרק שגיאה.

**התיקון:** ביטלתי לחלוטין את הגיבוי האוטומטי ל-localStorage - הוא לא נחוץ כי הכל נשמר ב-MySQL DB ממילא.
