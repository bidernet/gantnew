# 🎨 bidernet v2.3.8-php - מיתוג + אייקונים + UX

## 🆕 מה חדש ב-v2.3.8

### 🎨 מיתוג
1. **Favicon חדש** (10 גדלים) - הלוגו הירוק-לימון בכרטיסיית הדפדפן
2. **כותרת בדפדפן**: "בידרנט מערכת גאנטים"
3. **צבעי מותג**: `#d7ff00` (ירוק-לימון) + `#013d19` (ירוק כהה)
   - בכפתורי "פוסט חדש", "לקוח חדש", "מנהל חדש"
   - בטאבים הפעילים (ראשי / לפרסום / לקוחות / מנהלים)

### 📱 אייקוני פלטפורמות מותאמים
- **3 פלטפורמות בלבד**: Facebook, Instagram, TikTok (הוסרו X/Twitter ו-LinkedIn)
- **אייקונים ירוקים מותאמים** עם רקע שקוף
- **בעריכת פוסט**: כפתורי בחירת פלטפורמה - **רק אייקונים**, ללא טקסט

### 📝 UX משופר בעריכת פוסט
- **שדה תוכן הפוסט** - הוגדל מ-5 ל-**10 שורות** (כפול!) - יותר נוח לכתוב פוסטים ארוכים
- **אזור העלאת קובץ** - **מצומצם משמעותית** (שורה אחת קומפקטית במקום ריבוע גדול)

3. צור פוסט חדש → בחירת פלטפורמות - **רק 3 אייקונים** (Facebook, Instagram, TikTok)
4. שדה תוכן - **גדול יותר** (10 שורות)
5. אזור העלאת קובץ - **שורה אחת קומפקטית**
