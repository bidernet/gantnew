# 🚨 bidernet v2.9.9-php - תיקון קריטי!

## ⚠️ תיקון באג קריטי - אובדן תמונות

### הבעיה החמורה שהתגלתה
כשלקוח לחץ "אשר" על פוסט - **התמונה נמחקה מה-DB**!

### הסיבה הטכנית
מאז v2.9.3 (Lazy Loading), הפוסטים בזיכרון לא כללו את `mediaData`. כשהמערכת שמרה את הפוסט (עם האישור) לשרת - היא שלחה אותו **בלי תמונה** → השרת דרס את התמונה הקיימת עם תוכן ריק.

### התיקון
**כל מקום שיש בו שמירה לשרת** עכשיו מוודא קודם שיש mediaData בזיכרון. אם אין - טוען אותו מהשרת לפני השמירה.

תוקן ב-4 מקומות קריטיים:
1. ✅ **`updatePostApproval`** - אישור/דחיית פוסט על ידי לקוח
2. ✅ **`savePosts` (לקוח)** - שמירת פוסטים בצד הלקוח
3. ✅ **`onEditPost` (לקוח)** - עריכת פוסט בצד הלקוח
4. ✅ **`savePosts` (אדמין)** - עכשיו מגן גם על פוסטים ישנים בלי flags
