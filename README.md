# 🎉 bidernet v2.4.6-php - לקוח עורך ומבקש שינוי

## ✨ מה חדש ב-v2.4.6

### בצד הלקוח - 2 אופציות לפעולה
על כל פוסט יש עכשיו 3 כפתורים:

1. 🟢 **"ערוך פוסט"** - הלקוח עורך את המלל בעצמו
2. ⚪ **"בקש שינוי"** - הלקוח כותב הערה והאדמין יתקן
3. 🟢 **"אשר פוסט"** - אישור סופי (למטה)

### תיקוני באגים
- 🧹 **HTML stripping תקין** - בעריכה רואים טקסט נקי בלי `<div>` ו-`&nbsp;`
- 🎨 **הערת שינוי בולטת ברשימה** - בצבעי המותג ירוק-לימון
- 💾 **שמירה תקינה** - escape של תווים מיוחדים, שורות נשמרות נכון

### כשלקוח עורך פוסט:
1. הפוסט מתעדכן ב-DB
2. הסטטוס מתאפס ל"ממתין לאישור"
3. נכנס ל-history: "✏️ הלקוח ערך את תוכן הפוסט"
4. אישור: "הפוסט עודכן בהצלחה!"

## 🚀 התקנה

העלה רק את **`app.js`** ל-`/public_html/app.bidernet.co.il/`

**Ctrl+Shift+R** ב-Console (F12):

1. **כלקוח** - פתח פוסט
