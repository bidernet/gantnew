# 🃏 bidernet v2.7.0-php - תצוגת Grid חדשה!

## ✨ מה חדש בגרסה זו

### 🎴 תצוגת ציר הזמן עכשיו ב-Grid
במקום שורות ארוכות, הפוסטים מוצגים ככרטיסיות יפות בגריד.

**כל כרטיס מציג:**
- 🖼️ תמונה מוקטנת בראש (16:9)
- 🏷️ נקודת צבע + שם עסק
- 📝 כותרת הפוסט
- 📄 תיאור מקוצר (2-3 שורות)
- 📅 תאריך + ⏰ שעה
- 📘 פלטפורמות + סטטוס
- 🎨 צבע מסגרת לפי סטטוס:
  - 🟢 ירוק = אושר ע"י לקוח
  - 🟡 כתום = ממתין לאישור
  - 🔴 אדום = דחה הלקוח

### 📱 רספונסיבי
| גודל מסך | כרטיסים בשורה |
|---|---|
| מובייל (< 640px) | 1 |
| טאבלט (640-1024px) | 2 |
| מחשב (1024-1280px) | 3 |
| מחשב גדול (> 1280px) | 4 |

### 🎯 פיצ'רים בכרטיס אדמין
- **Hover** - הכרטיס מתרומם עם צל גדול יותר
- **Action buttons** - בלחיצה על hover: ערוך / מחק / שכפל
- **תמונת placeholder** - אם אין תמונה, אייקון של הקטגוריה ברקע צבעוני עדין
