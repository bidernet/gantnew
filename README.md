# 📅 bidernet group - מערכת ניהול גאנטי תוכן v2.1.7

## 🆕 שינוי בגרסה זו (24/05/2026)

📅 **תצוגה חודשית בציר זמן** - גם המנהל וגם הלקוח רואים בברירת מחדל רק את **החודש הנוכחי**

### איך זה עובד?
1. כשנכנסים למערכת בציר זמן (Timeline) - רואים רק את פוסטי **החודש הנוכחי**
2. מעל הרשימה יש חיצי ניווט `< >` לדפדף בין חודשים
3. כפתור "היום" קופץ חזרה לחודש הנוכחי
4. ההסתרה היא **רק בציר הזמן** - בלוח שנה ובגאנט עדיין יש את הניווט שלהם

### למה זה חשוב?
- 🎯 **המנהל**: ממוקד בחודש הנוכחי, פחות עומס מידע על המסך
- 👁️ **הלקוח**: רואה רק מה רלוונטי לו עכשיו, לא רואה היסטוריה ישנה ולא רואה חודש שעוד לא הוכן
- 🗓️ **שני הצדדים** יכולים לחזור לחודשים קודמים אם צריך - הכל זמין, רק חבוי בברירת מחדל

### תרחיש לדוגמה
- היום: **24 במאי 2026**
- ברירת מחדל: רואה רק את פוסטי **מאי 2026**
- רוצה לראות אפריל? לוחץ `<` → רואה את אפריל
- רוצה להכין יוני (כמנהל)? לוחץ `>` → רואה ועובד על יוני
- חוזר עם "היום" → חזר למאי
