# 📋 bidernet v2.9.0-php - שאלוני הצטרפות לקוח

## 🎉 פיצ'ר ענק חדש!

### 📋 מערכת שאלוני הצטרפות מלאה
מערכת מקיפה לבניית שאלונים ואיסוף מידע מלקוחות חדשים.

#### 🆕 לשונית חדשה ב-Sidebar: **"שאלונים"** 📋

##### 📑 טאב 1: **שאלונים**
- כל השאלונים שיצרת מוצגים ב-Grid
- כפתור "+ שאלון חדש" ליצירה מאפס
- **תבנית מוכנה לשאלון הצטרפות לקוח** (לפי הדוגמה שלך)
- לכל שאלון:
  - 📋 **העתק לינק** - לינק לשליחה ללקוח
  - ✏️ **עריכה** - שינוי שאלות וסעיפים
  - 🗑️ **מחיקה**

##### 💬 טאב 2: **תשובות שהתקבלו**
- רשימה של כל התשובות
- ▶️ סטטוס: חדש (אדום) / נקרא / **הומר ללקוח** (ירוק)
- לחיצה → רואים את כל התשובות
- **כפתור "ייצור לקוח"** - יוצר משתמש לקוח אוטומטית במערכת!

#### 🔨 עורך שאלונים מלא
- הוסף סעיפים (sections)
- בכל סעיף הוסף שאלות עם 7 סוגים:
  - 📝 טקסט קצר
  - 📄 טקסט ארוך
  - 📞 טלפון
  - 📧 אימייל
  - 🎴 בחירה ויזואלית (cards עם אייקונים)
  - ☑️ בחירה מרובה (chips)
  - 📋 רשימה נפתחת
- אפשרות לסמן כל שאלה כ"חובה"
- רמז עזרה לכל שאלה

#### 🔗 שליחת שאלון ללקוח
1. צור שאלון
2. העתק את הלינק (כפתור 📋)
3. שלח ללקוח בוואטסאפ/מייל
4. הלינק נראה ככה: `https://app.bidernet.co.il/?q=onboarding_xxx`

#### 👤 דף השאלון של הלקוח
- עיצוב יפה עם הלוגו שלך
- Progress bar מתעדכן בזמן אמת
- ולידציה של שדות חובה
- הודעת "תודה רבה" אחרי שליחה

#### 🎯 המרה ללקוח
מתשובה שהגיעה לחץ "ייצור לקוח" - המערכת יוצרת לבד:
- משתמש חדש עם שם המשתמש = האימייל/טלפון
- סיסמה אוטומטית
- מעבירה את שם העסק ופרטי הקשר
- מסמנת את התשובה כ"הומר"

## 🚀 התקנה

### צעד 1: גיבוי 🛡️
phpMyAdmin → Export → Go

### צעד 2: העלאה ל-`/public_html/app.bidernet.co.il/`
- ✅ **`app.js`** ⭐
- ✅ **`api.php`** ⭐
- ❌ אל תחליף `config.php`

### צעד 3: בדיקה
ה-API **אוטומטית** יוצר את הטבלאות החדשות בפעם הראשונה.
לא צריך להריץ schema.sql ידנית!

**Ctrl+Shift+R** ב-Console (F12):
```
🎯 bidernet Content Calendar v2.9.0-php
📋 NEW: Client onboarding questionnaires
```

## 🧪 איך לבדוק

1. **היכנס כמנהל** → לחץ "📋 שאלונים" בסיידבר
2. **לחץ "🚀 התחל מתבנית"** - תפתח שאלון הצטרפות מוכן
3. **שמור**
4. **לחץ על "לינק"** - העתק את הלינק
5. **פתח את הלינק בטאב פרטי** - תראה את השאלון מנקודת מבט של לקוח
6. **מלא ושלח**
7. **חזור לאדמין** → "תשובות שהתקבלו" → תראה את התשובה!
8. **לחץ על התשובה** → תראה הכל
9. **"ייצור לקוח"** - יוצר משתמש לקוח אוטומטית
