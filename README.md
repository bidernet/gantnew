- 🔍 שורת חיפוש קומפקטית

## 🛠 טכנולוגיות

- React 18 + Vite
- Tailwind CSS
- lucide-react icons
- localStorage (פיתוח) / API + MySQL (ייצור)

## 🚀 התקנה

```bash
# התקן תלויות
npm install

# פיתוח
npm run dev
# פותח http://localhost:5173

# בנייה לייצור
npm run build
# מייצר תיקיית /dist להעלאה לשרת
```

## 🌐 העלאה לשרת

### עם Vercel / Netlify (קל ביותר)

1. חבר את הריפוזיטורי שלך מ-GitHub
2. בחר Framework: **Vite**
3. Build Command: `npm run build`
4. Output Directory: `dist`
5. דחוף - וזה רץ אוטומטית

### עם VPS/שרת רגיל

```bash
npm install
npm run build
# העתק את תיקיית /dist לשרת
# שרת אותה עם nginx/apache
```
