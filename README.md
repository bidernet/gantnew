# 📅 bidernet group - מערכת ניהול גאנטי תוכן v2.1.5

## 🆕 שינוי בגרסה זו (24/05/2026)

⚙️ **כפתור "מיתוג" בהדר** - עכשיו תוכל לעדכן את הלוגו והצבעים של המערכת בעצמך!

### איך זה עובד?
1. בהדר העליון, ליד כפתור "יציאה", יש כפתור חדש **⚙️ מיתוג**
2. לחץ עליו - יפתח חלון הגדרות מיתוג
3. שם תוכל:
   - 📸 **להעלות לוגו חדש** (PNG/JPG עד 2MB)
   - 🎨 **לשנות צבעים ראשיים ומשניים**
   - ✏️ **לעדכן שם חברה וסלוגן**
   - 🔄 **לאפס לברירת מחדל** (יחזיר את הלוגו של bidernet group)
4. לחץ "שמור" - השינויים יופיעו מיד בכל המערכת

### יתרון גדול
**עכשיו אינך תלוי בי או בקובץ הקוד** - תוכל לעדכן את הלוגו, הצבעים והמיתוג בכל זמן ובאופן עצמאי לחלוטין דרך הממשק עצמו! 🚀

## 🚀 התקנה והפעלה

```bash
# התקנה
npm install

# פיתוח (לעבודה לוקלית)
npm run dev

# בנייה לייצור
npm run build
# יוצר תיקיית /dist להעלאה לשרת
```

## 📋 איך לוודא שהגרסה החדשה רצה

1. רענן עם **Ctrl+Shift+R** (Windows) או **Cmd+Shift+R** (Mac)
2. פתח **DevTools** (F12) → **Console**
3. אמור להופיע:
   ```
   🎯 bidernet Content Calendar v2.1.5
   ✓ Branding Settings modal added to admin header
   ```
4. בהדר העליון תראה כפתור חדש **⚙️ מיתוג**

## 🎨 איך להחליף את הלוגו (מדריך מהיר)

1. היכנס כ-admin
2. לחץ על **⚙️ מיתוג** בהדר
3. לחץ "החלף לוגו" → בחר קובץ PNG/JPG (עדיף PNG עם רקע שקוף)
4. לחץ **שמור**
5. הלוגו יופיע מיד בכל המקומות: התחברות, הדר מנהל, הדר לקוח

💡 **טיפ**: אם הלוגו שלך עם רקע שחור או צבעוני, השתמש ב-https://www.remove.bg/ כדי להסיר את הרקע ולקבל PNG שקוף.
