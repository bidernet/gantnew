# 🎠 bidernet v2.8.0-php - קרוסלה + 4:5

## ✨ מה חדש בגרסה זו

### 📐 1. Aspect Ratio 4:5 (Instagram Standard)
הכרטיסים עכשיו ב-4:5 - **תמונת הפוסט מוצגת בשלמותה** בלי להיחתך.
- הגודל הסטנדרטי של פייסבוק/אינסטגרם
- בצד האדמין ובצד הלקוח

### 🎠 2. תמיכה בקרוסלה (עד 10 תמונות)
**בעת יצירת פוסט** - בוחרים בין:
- 📄 **פוסט רגיל** - תמונה אחת או וידאו (כמו עכשיו)
- 🎠 **קרוסלה** - עד 10 תמונות

**במצב קרוסלה:**
- העלאה מרובה של תמונות (multi-select)
- thumbnails של כל התמונות בגריד
- חיצים לסידור מחדש (←/→)
- כפתור מחיקה לכל תמונה
- מונה X / 10

**בכרטיס:**
- מציג את התמונה הראשונה
- Badge קטן בפינה: 📋 1/5

**במודאל מלא:**
- חיצים גדולים ימין/שמאל
- נקודות ניווט בתחתית
- מונה "3 / 5" בפינה

## 🚀 התקנה

### צעד 1: גיבוי 🛡️
לפני שאתה עושה משהו - **תעשה גיבוי מ-phpMyAdmin!**

### צעד 2: הרץ migration ב-DB
פתח phpMyAdmin → SQL → הדבק את התוכן של `schema.sql` → **Go**

זה ירוץ migrations שמוסיפים שני שדות חדשים (`isCarousel`, `mediaItems`) ל-DB.
**בטוח** - מריץ ALTER TABLE רק אם השדות לא קיימים.

### צעד 3: העלה ל-cPanel
ל-`/public_html/app.bidernet.co.il/`:
- ✅ **`app.js`** (חובה)
- ✅ **`api.php`** (חובה - כולל את שינויי השדות החדשים)
- ❌ **אל תחליף** את `config.php`
