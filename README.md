## 🚀 התקנה והפעלה

```bash
# התקנה
npm install

# פיתוח (לעבודה לוקלית)
npm run dev

# בנייה לייצור
npm run build
# יוצר תיקיית /dist להעלאה לשרת
```

## 📋 איך לוודא שהגרסה החדשה רצה

1. רענן עם **Ctrl+Shift+R** (Windows) או **Cmd+Shift+R** (Mac)
2. פתח **DevTools** (F12) → **Console**
3. אמור להופיע:
