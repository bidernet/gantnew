## 📋 איך לוודא שהגרסה החדשה רצה

1. רענן עם **Ctrl+Shift+R** (Windows) או **Cmd+Shift+R** (Mac)
2. פתח **DevTools** (F12) → **Console**
3. אמור להופיע:
   ```
   🎯 bidernet Content Calendar v2.1.3
   ✓ Category removed from post modal
   ```

## 🚀 התקנה והפעלה

```bash
# התקנה
npm install

# פיתוח (לעבודה לוקלית)
npm run dev
# פותח: http://localhost:5173

# בנייה לייצור
npm run build
# יוצר תיקיית /dist להעלאה לשרת
```

## 🌐 העלאה לשרת

### Vercel / Netlify (מומלץ):
1. חבר את הריפוזיטורי שלך מ-GitHub
2. Framework: **Vite**
3. Build Command: `npm run build`
4. Output Directory: `dist`

### שרת VPS רגיל:
```bash
npm install
npm run build
# העלה את תיקיית /dist לשרת
```
